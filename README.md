STM32F103C8T6 Custom Development Board

This repository contains the open-source hardware design for a custom STM32-based embedded development board, designed for IoT and sensor integration.

Project Overview

This board was developed to provide a compact, versatile platform for ARM Cortex-M based projects. It features an integrated USB-to-UART bridge, bootloader logic for firmware flashing, and optimized power management.  

Key FeaturesMicrocontroller: STM32F103C8T6 (ARM Cortex-M3).  

Connectivity: Integrated FT232RL USB-to-UART bridge.  

Power Management: AMS1117-3.3V voltage regulator.  

Storage: AT24C256C Serial EEPROM for data logging.  

Debug/Flash: Standard JTAG/SWD interface.
